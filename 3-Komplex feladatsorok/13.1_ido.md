# Kapuzárásig hátralévő idő

Egy repülőtéren az utasoknak figyelniük kell, hogy mikor zár a beszállókapu. A kapu zárása után már nem lehet felszállni a repülőgépre.

Készítsen Python programot, amely bekéri a felhasználótól:

```txt
jelenlegi idő
kapuzárás ideje
```

Mindkét időpontot az alábbi formátumban kell megadni:

```txt
óó:pp
```

Például:

```txt
08:35
09:10
```

A program számítsa ki, hogy mennyi idő van még hátra a kapuzárásig!

Az eredményt **óra és perc** formában jelenítse meg.

Ha a kapuzárás ideje már elmúlt, akkor a program írja ki:

```txt
A kapu már bezárt.
```

Ha még nem múlt el a kapuzárás ideje, akkor például így jelenjen meg az eredmény:

```txt
A kapuzárásig még 0 óra 35 perc van hátra.
```

## Példa futás

```txt
Adja meg a jelenlegi időt: 08:35
Adja meg a kapuzárás idejét: 09:10
A kapuzárásig még 0 óra 35 perc van hátra.
```

## Másik példa futás

```txt
Adja meg a jelenlegi időt: 09:20
Adja meg a kapuzárás idejét: 09:10
```
