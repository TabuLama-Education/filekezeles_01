# **Feladatleírás**

Egy iskola tanulóiról egy `tanulok.txt` fájl tartalmaz adatokat.  
Minden sorban egy tanuló neve, életkora és átlaga szerepel pontosvesszővel elválasztva.

###  `tanulok.txt`

```
Anna;17;4.2
Béla;16;3.7
Csaba;18;4.8
Dóra;17;3.9
Erika;16;4.1
```

###  Feladatok

1. Olvasd be a fájl tartalmát.
    
2. Tárold az adatokat egy listában.
    
3. Írd ki a tanulók számát.
    
4. Számold ki az átlagéletkort.
    
5. Írd ki a legjobb tanuló nevét és átlagát.
    
6. Mentsd új fájlba (`jo_tanulok.txt`) azok nevét, akiknek az átlaga legalább **4,0**.
    


##  **Program futásának eredménye**

```
Tanulók száma: 5
Átlagéletkor: 16.8
Legjobb tanuló: Csaba - 4.8
A 'jo_tanulok.txt' fájl elkészült.
```

 **`jo_tanulok.txt` tartalma:**

```
Anna
Csaba
Erika
```
